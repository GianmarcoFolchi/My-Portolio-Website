Reunite App

React Native

As a participant in Knight Hacks 2021, I worked in a team of four to develop a React Native mobile app which would allow users to create social events with specific tags and allow them to view all nearby events which align with their interests. As the backend developer, I set up a Firebase database and a Node server based on Express. Additionally, I created the client side functions which communicate the frontend and the API. The development of this project taught me how to create Restful APIs and how to store and retrieve data from databases.
