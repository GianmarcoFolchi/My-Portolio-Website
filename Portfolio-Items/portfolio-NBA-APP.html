Guess the NBA Player App

iOS App

I am the sole developer of The Guess the NBA Player app. The premise of the app is to guess the NBA player given their stats. This app is built using Swift and Xcode with the SwiftUI framework for the frontend. The backend is built on the "balldontlie" API and a Python script to scrape NBA player names. Creating this project enhanced my app development skills while I learned to work with JSON and adapted to the SwiftUI framework.
